# 🤖 AI Agent med AgentScope

Dette er en lokal AI-agent bygget med AgentScope og Ollama (Llama 3).

## 🚀 Features

- 💬 Chat med AI
- 🌐 Søker på internett (DuckDuckGo)
- 💻 Genererer og kjører Python-kode
- 💾 Lagrer filer automatisk

## 🧠 Hvordan det funker

Agenten:
1. Leser input fra bruker
2. Bestemmer om den skal:
   - svare
   - søke på nett
   - skrive kode
3. Kjører handlingen
